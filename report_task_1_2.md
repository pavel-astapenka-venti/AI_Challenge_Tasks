# Task 2: How I did this

Used Lovable to create app + GitHub Copilot to analyze requirements generate test cases + Playwright MCP to automatically test app. First I created MVP just prompting requirements as is, result was really buggy. Then I used GitHub Copilot to create test cases based on requirements and Playwright MCP to test by them. Any bugs that were found were prompted to fix and tested again, until fixed and verified.
